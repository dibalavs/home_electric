Проект электроснабжения моей новой квартиры в ЖК Скандинавия.

# Подготовка

Ubuntu 22.04

```
sudo apt update
sudo apt install pandoc weasyprint
```

# Компиляция

Генерация pdf файла в каталоге output:
```
./gen_output.sh
```

Генерация html файла в каталоге output:
```
./gen_output.sh html
```
